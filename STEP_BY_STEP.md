## 步骤 8️⃣: 安装到手机

### 传输 APK
```
方式 1: 数据线 + adb install
方式 2: 微信/QQ 发送到手机
方式 3: 扫码直接下载
方式 4: 云盘传输
```

### 安装过程
```
1. 在手机上找到 APK 文件
   ↓
2. 点击安装
   ↓
3. 如果提示"未知来源"
   → 设置 → 安全 → 允许安装未知应用
   ↓
4. 继续安装
   ↓
5. 安装完成 ✅
```
