# 快速参考卡片

## 🚀 5 分钟打包速查

### 📥 下载 HBuilderX
```
https://www.dcloud.io/hbuilderx.html
选择：App 开发版
```

### 📂 导入项目
```bash
git clone https://github.com/iwenke/lazy-money.git
```
拖拽 `lazy-money` 文件夹到 HBuilderX

### 📦 打包
```
菜单: 发行 → 原生 App-云打包
平台: ☑️ Android
证书: 使用自有证书
```

### 🔑 证书信息
```
文件: lazy-money.keystore
别名: lazy-money
密码: android123
```

### ⏳ 等待
```
5-10 分钟
```

### 📥 下载
```
点击"下载"按钮
安装到手机测试
```

## 🔧 故障排查

| 问题 | 解决方案 |
|------|----------|
| 打包失败 | 检查网络、查看控制台错误 |
| 证书错误 | 确认密码是 `android123` |
| 无法安装 | 允许"安装未知来源应用" |
| 签名不一致 | 使用相同证书或卸载旧版 |

## 🎯 关键提示

✅ **第一次打包**
- 使用"DCloud 公用证书"快速测试
- 确认功能正常后再用自有证书

✅ **正式发布**
- 必须使用"自有证书"
- 证书文件妥善保管
- 密码不要忘记

✅ **更新版本**
- 修改版本号
- 使用相同证书
- 可以覆盖安装

⚠️ **重要警告**
- 证书丢失无法更新应用
- 不同证书无法覆盖安装
- 公用证书无法发布到商店
