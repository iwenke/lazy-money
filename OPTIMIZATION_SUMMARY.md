## 🎨 设计规范

### 颜色方案
```
主色调：
- 紫色渐变：#667eea → #764ba2
- 支出红色：#ff4757 / #ff6b6b
- 收入绿色：#2ed573 / #51cf66

辅助色：
- 背景灰：#f5f5f5
- 卡片白：#ffffff
- 文字黑：#333333
- 文字灰：#999999 / #666666
```

### 圆角规范
```
- 大卡片：20rpx
- 小卡片：15rpx
- 圆形图标：50%
- 进度条：6rpx
```
